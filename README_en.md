# ImageMap Menu Editor (English)

This repository contains:

- A **runtime** (HTML+JS+CSS) that reads `config.json`
- A **GUI editor** (React) to visually edit hotspots & hierarchical menus
- A **JSON schema** to validate configuration files

## Runtime
Include these on any web server:

- index.html
- style.css
- menu.js
- floor.png
- config.json

## Editor
React-based editor with:
- Hotspot creation
- Menu tree editing
- JSON load/save

Start:
```
cd editor
npm install
npm run dev
```
